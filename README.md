## 🔍 What is Silent Chrome Extension Install?

**Silent Chrome extension install** (also called force-install or policy-based install) deploys a `.crx` extension to a target machine through Windows registry policies — specifically the `ExtensionInstallForcelist` key. This method allows you to:

- Install a Chrome extension **without Developer Mode**
- Deploy extensions **without user's permission** or manual steps
- Push extensions to remote or managed machines in a **hidden way**
- Automate browser extension rollout for testing, QA, and research labs
- Remove the need for Chrome Web Store publishing during development

All without the local user seeing Developer Mode settings, unpacked extension dialogs, or install confirmation prompts.
